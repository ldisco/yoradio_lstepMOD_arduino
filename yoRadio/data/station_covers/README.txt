# Директория для кэша обложек

Эта папка используется для кэширования обложек радиостанций.
Обложки загружаются автоматически из iTunes API и сохраняются здесь для быстрого доступа.

Структура:
- cache/ - папка с кэшированными файлами обложек  
- cache_index.txt - индексный файл для быстрого поиска

Файлы создаются автоматически при работе устройства.
